Эти сигналы можно отправлять с помощью команды kill . Например, kill PID завершит процесс с указанным идентификатором. По умолчанию kill отправляет сигнал SIGTERM. 
Если использовать kill (без опций) по отношению к процессу, который был приостановлен при помощи Ctrl+Z, то процесс приступит к завершению, как только будет продолжен.
Чтобы отправить SIGKILL необходимо дописать -9. 
```
kill -9 9898
```
Такой способ завершения процесса является мгновенным, но небезопасным и применяется в крайних случаях. 
killall — команда, которая прерывает процессы с одним и тем же названием.
